# Design Rules

## Rules
- Mantener jerarquía visual clara (títulos > subtítulos > texto)
- Usar máximo 2 tipografías (una para títulos y otra para contenido)
- Evitar colores saturados o neón
- Usar paletas coherentes con buen contraste
- Bordes redondeados modernos
- Sombras suaves y sutiles (evitar sombras duras)

## Spacing & Layout
- Usar sistema de espaciado consistente
- Evitar márgenes aleatorios
- Alinear todo a un grid o contenedor central
- Mantener whitespace para respiración visual

## Responsive Design
- Mobile-first approach siempre
- Diseñar primero para 360px–480px
- Breakpoints claros (sm, md, lg)
- Evitar widths fijos cuando sea posible

## Accessibility
- Mantener contraste AA mínimo
- Estados hover y focus visibles
- No depender solo del color para información
- Botones con tamaño mínimo táctil (44px)

## UI Behavior
- Animaciones suaves
- Transiciones con easing natural
- Feedback visual en interacciones (hover/click/loading)

## CSS Practices
- Evitar !important
- Reutilizar clases en lugar de duplicar estilos
- Preferir variables CSS (--primary-color)
- Mantener estructura modular
